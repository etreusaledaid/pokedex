# pokedex
App para jugar ¿Quién es ese pokemón?

## Instalación

Utiliza la dependencia [Kingfisher](https://swiftpackageindex.com/onevcat/Kingfisher) para tratar las imagenes.

Para usarlo solo debe importar la dependencia
```bash
import Kingfisher
```
